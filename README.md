# PDFTeller

PDFTeller is my idea for a PDF reader with **text-to-speech (TTS)** support.  
While a sentence is spoken by the TTS, the same sentence is simply displayed.  

## Features
- **Sentence-by-sentence reading** with TTS.  
- **Library function** for storing processed PDFs (sentence-to-sentence).  
- **Checkpoint** to resume from the last read sentence.  
- **Dyslexia-friendly theme** with accessible font and colors.  

## Implementations
- **Python**: working version.  
- **Kotlin**: also written in Kotlin, since I thought it would make porting to Android easier — but all attempts were unsuccessful.
